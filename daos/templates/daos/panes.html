Pane Analysis of the DAOs Researched


On this page you will find a "10-pane" approach for the in-depth analyis conducted on each of the Decentralized Autonomous Organizations (DAOs). Each DAO has its own respective pane which consists of a network graph and various calculations below the network graph. The network graph is created by querying transaction data for a DAOs respective coin over a certain interval of time. The nodes in each chart correspond to the wallet addresses (both to and from) and the edges correspond to the transaction itself. Each edge contains the data-time stamp of the transaction, as well as, the amount of the respective token involved in the transaction. You may notice that some nodes are colored differently than the others in the chart, this is because I highlighted the Decentralized Exchanges (DEXs) and the DAO-specific contracts in order to develop a better understanding of how much of the typical transactions are peer-to-peer and fully decentralized. Furthermore, you may notice that some transactions are for zero coins, this is not an error as there are possible ways to engage in a transaction with zero coins. This type of transaction typically occurs in interactions between contracts or peers and contracts.
